## About
The Sudoku Solver is a web-based application built with **HTML, CSS, and JavaScript**. It allows users to solve Sudoku puzzles instantly using a **backtracking algorithm** or play interactively with **timer, difficulty levels, error highlighting, and responsive design**.

---

## Features
- Solve Sudoku puzzles automatically.
- Play mode with **Easy, Medium, and Hard** levels.
- Real-time **error highlighting** for invalid moves.
- **Timer** and **mistake counter** for challenge tracking.
- Responsive **grid design** using CSS Grid.
- **Win modal popup** upon successful completion.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Algorithm:** Backtracking
- **Design:** Responsive CSS Grid Layout
